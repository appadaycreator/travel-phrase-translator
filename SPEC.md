# 海外旅行フレーズ変換ツール【無料】無料Webツール - 技術仕様書

## 概要

**サービス名**: Travel Phrase Translator
**バージョン**: 2.3.0（利用統計・印刷・エクスポート機能追加）
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/travel-phrase-translator/

7シーン×6言語のフレーズを自動表示。コピーボタン・音声読み上げ・キーワード検索つき。登録不要・完全無料。

## 対応シーン（v2.0 追加）

| シーン | フレーズ数 |
|-------|---------|
| ✈️ 空港・機内 | 12件 |
| 🏨 ホテル | 12件 |
| 🍽️ レストラン | 12件 |
| 🛍️ ショッピング | 12件 |
| 🚨 緊急・トラブル | 12件 |
| 🚌 交通・移動（NEW） | 12件 |
| 🏛️ 観光・体験（NEW） | 12件 |

## 対応言語（v2.0 追加）

英語 / 中国語（簡体字）/ 韓国語 / タイ語 / スペイン語（NEW）/ フランス語（NEW）

## 新機能（v2.0）

- **自動表示**: ページ読み込み・セレクト変更で即座にフレーズ表示（ボタン不要）
- **コピー機能**: 📋ボタンで外国語フレーズをクリップボードにコピー
- **音声読み上げ**: 🔊ボタンでWeb Speech APIによる発音確認
- **キーワード検索**: 外国語・日本語訳の両方で絞り込み検索
- **件数表示**: 表示中フレーズ数をリアルタイム表示

## 新機能（v2.1.0 - 2026-06-03）

- **Service Worker オフライン対応**: 電波不安定な環境でも利用可能
- **フレーズお気に入い**: ⭐ボタンでよく使うフレーズを登録（localStorage保存）
- **拡張コピー機能**: 右クリックで「フレーズのみ」or「フレーズ + 日本語訳」を選択
- **ダークモード**: 夜間使用向け。システム設定に自動対応し、手動トグルも可能

## 新機能（v2.3.0 - 2026-06-10）

- **利用統計グラフ表示**: Chart.js でコピー・お気に入い数を可視化（TOP3シーン）
- **印刷機能**: 🖨️ ボタンでフレーズ一覧を紙出力用に最適化して印刷
- **テキスト一括コピー**: 「すべてコピー」でシーン全体のフレーズリストをコピー
- **CSVエクスポート**: 📥 ボタンでフレーズ一覧を Excel 形式でダウンロード

## データ管理

- **ストレージ**: なし（全フレーズをJSに内蔵、外部API通信なし）
- **音声**: Web Speech API（ブラウザ内処理）

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript（ES5互換）
- PWA対応（manifest.json）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. シーン（空港・ホテルなど）と言語を選ぶ
2. フレーズ一覧が自動表示される
3. 📋ボタンで外国語フレーズをコピー
4. 🔊ボタンで発音を音声確認
5. 旅行先で相手に見せるか読み上げて活用する
